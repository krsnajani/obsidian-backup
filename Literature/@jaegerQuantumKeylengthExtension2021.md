<!-- START_ABSTRACT -->
Should quantum computers become available, they will reduce the eﬀective key length of basic secret-key primitives, such as blockciphers. To address this we will either need to use blockciphers with inherently longer keys or develop key-length extension techniques to amplify the security of a blockcipher to use longer keys.
